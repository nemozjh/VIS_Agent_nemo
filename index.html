Project Overview

We present Structured AI Agents, a modular pipeline for visualization report generation. The task is decomposed into sub-task units that yield standardized JSON with executable code. During execution, quantitative results are deterministically computed and chart rendering is reproducible, reducing hallucinations and improving reliability in the final report.

Abstract

This work addresses instability and hallucination in AI-agent-based data visualization by introducing a structured pipeline. Each sub-task unit produces standardized JSON that embeds executable and deterministic code. The design enforces correctness in quantitative computation, reduces hallucinations, and yields reliable visualizations. Verified descriptions and generated charts are then composed into a coherent, reproducible report.
